line-height

View SCSS source

Apply line-height values with utility classes. Values can optionally be applied only at specific breakpoints.

Using the Sass mixin is preferred, as it forces us to consider all objects as components. Applying utility classes in HTML enforces !important, as they're intended to be immutable â€” applying one property and value no matter what.

Values

  • default
  • xx-tight
  • x-tight
  • tight
  • loose
  • x-loose
  • xx-loose
  • 1 to 11 (only available in mixin)
